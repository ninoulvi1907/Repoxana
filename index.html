💻 GitHub nədir?

GitHub proqramçıların kod paylaşdığı, layihələri birlikdə inkişaf etdirdiyi və open-source dünyasının mərkəzi hesab olunan platformadır.

Burada:

• Kod yaza bilərsən
• Layihələri host edə bilərsən
• Başqa developer-lərin reposuna baxa bilərsən
• Open-source layihələrə contribution edə bilərsən
• Öz portfolio-nu qura bilərsən
• Minlərlə faydalı tool və AI layihəsi tapa bilərsən

GitHub həm də Git adlı version control sisteminin üzərində qurulub. Bu sayədə kod dəyişikliklərini izləmək, komandada işləmək və layihələri idarə etmək çox rahat olur 🚀

Bu gün istifadə etdiyin bir çox məşhur texnologiya:

• React
• Next.js
• TensorFlow
• Linux
• VS Code

hamısı GitHub üzərində open-source şəkildə inkişaf etdirilir.

📦 Repository nədir?

Repository (repo) layihənin bütün kodlarının, fayllarının və versiyalarının saxlanıldığı yerdir.

Sadə dildə: repo = layihənin əsas qovluğu 💙

Burada:

• Kod faylları saxlanılır
• README və dokumentasiya yerləşir
• Şəkillər və digər resurslar olur
• Commit tarixçəsi saxlanılır
• Layihənin bütün dəyişiklikləri izlənilir
• Developer-lər birlikdə işləyə bilir

Məsələn:

• Bir website
• Mobil tətbiq
• AI layihəsi
• Discord botu
• API sistemi

hamısı bir repository içində yerləşə bilər 🚀

GitHub üzərində gördüyün hər layihə əslində bir repository-dir. Məsələn:

• React
• Next.js
• TensorFlow
• Linux

bunların hamısının öz reposu var 💻
